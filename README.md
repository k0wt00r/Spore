## ⚠ Troubleshooting

- **App does not start:** Make sure your Windows is updated and you have administrator rights.
- **Installation fails:** Try running the installer as an administrator (right-click the file > Run as administrator).
- **AI tasks stop unexpectedly:** Restart the app and clear running tasks.
- **Slow performance:** Close other programs and check for background updates.
